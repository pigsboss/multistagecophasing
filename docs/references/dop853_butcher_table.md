# DOP853 (Dormand‑Prince 8(5,3)) Butcher Tableau

## Description

The DOP853 integrator implemented in `mission_sim/utils/propagators/rk.py` is the **8(5,3)** embedded pair due to Dormand & Prince, as published in Hairer, Nørsett & Wanner (1993).  
The coefficients are identical to those of the classical Fortran 77 code `dop853.f` by Hairer & Wanner.

- Number of stages: 12  
- Order of high‑order solution: 8  
- Order of embedded error estimate: 5 (and an additional 3rd‑order estimate used for step‑size stabilisation)  
- FSAL: **No** (the last stage is **not** reused).

## Notes

- The low‑order weights (5th order) can be obtained as `B_low = B_high - ε`.
- The dense output coefficients D4–D7 and the three extra stages (14–16) are required only when `IOUT ≥ 2`. They are used to construct the continuous solution on the interval `[x, x+h]`.
- All constants in this document are transcribed directly from `mission_sim/utils/propagators/dop853.f` by Hairer & Wanner.
- The Python implementation in `rk.py` contains the same values, with the dense output matrices scoped to `_dop853` only when needed (the code stores `CONT` for the dense output components).

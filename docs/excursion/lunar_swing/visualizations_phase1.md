# 第一阶段可视化产出规划

> **目标**：为第一阶段（概念设计与基础设施准备）规划关键的可视化产出，用于澄清概念、展示设计，并为后续开发提供视觉参考。

## 1. 摆动轨道机制示意图

### 1.1 内容要点
- 展示轨道在地球和月球之间的"摆动"运动
- 标注关键点：近地点（大气穿越）、远地点（月球引力辅助）
- 显示能量流动：大气阻力损失 ↔ 月球引力增益
- 标注共振比概念（如2:1）

### 1.2 技术规格
- **格式**：矢量图（SVG）或高清PNG
- **工具**：draw.io / diagrams.net 或 Python matplotlib
- **交付物**：`diagrams/swing_mechanism.png`
- **用途**：项目介绍、技术文档插图

## 2. CRTBP坐标系与零速度面示意图

### 2.1 内容要点
- 展示地月旋转坐标系：原点在地月质心，x轴指向月球，z轴垂直轨道面
- 显示五个平动点（L1-L5）的位置
- 绘制特定雅可比常数下的零速度面（Zero Velocity Curves）
- 用不同颜色区分允许运动区和禁止运动区
- 在图上标注一条示例的摆动轨道轨迹

### 2.2 技术规格
- **格式**：3D渲染图（可交互HTML或静态PNG）
- **工具**：Python (matplotlib 3D 或 plotly)
- **交付物**：`diagrams/crtbp_coordinates.png` 和 `diagrams/crtbp_zvc.html`
- **用途**：理论文档插图、算法理解辅助

## 3. MCPC模块依赖关系图

### 3.1 内容要点
- 展示 lunar-swing 项目引入的新模块（`HighPrecisionEphemeris`, `UniversalCRTBP`等）
- 显示这些新模块与现有MCPC核心模块的依赖关系
- 用不同颜色区分：新模块、现有模块、接口契约
- 展示数据流方向

### 3.2 技术规格
- **格式**：架构图（使用标准UML或框线图）
- **工具**：draw.io / PlantUML
- **交付物**：`diagrams/module_dependencies.png`
- **用途**：架构设计文档、开发指引

## 4. 项目阶段与Sprint路线图

### 4.1 内容要点
- 可视化展示四个阶段（概念设计、算法原型、高保真实验、框架集成）
- 每个阶段包含的Sprint（1-2, 3-4, 5-6, 7-8）
- 标注每个Sprint的关键产出（文档、代码、测试、可视化）
- 显示阶段之间的依赖关系

### 4.2 技术规格
- **格式**：时间线图或甘特图
- **工具**：mermaid.js（已用于README）、draw.io
- **交付物**：`diagrams/project_timeline.png`
- **用途**：项目进度跟踪、团队同步

## 5. 桩测试覆盖率报告（初始基准）

### 5.1 内容要点
- 显示第一阶段结束时测试目录的初始结构
- 展示每个新模块接口对应的桩测试文件
- 标注待实现的测试用例数量
- 为后续测试覆盖率增长提供基准线

### 5.2 技术规格
- **格式**：树状结构图或表格
- **工具**：Python生成文本图表或markdown表格

- **用途**：测试管理、质量跟踪

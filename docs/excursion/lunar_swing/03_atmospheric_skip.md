# 03. 大气穿越技术：建模、仿真与验证

> **摘要**：本文档阐述地月共振摆动轨道中的大气穿越关键技术，包括大气密度建模、气动力计算、热效应分析以及穿越过程的数值仿真方法。为第三阶段（高保真数字实验）提供技术指导，确保轨道在真实大气摄动下的长期稳定性。

## 3.1 大气穿越在摆动轨道中的作用

### 3.1.1 能量平衡机制
- 大气阻力在近地点消耗轨道能量，降低远地点高度
- 月球引力在远地点提供能量增益，补偿大气损失
- 需要决策：能量平衡的量化目标（如ΔE_drag ≈ ΔE_moon ± 5%）

### 3.1.2 技术挑战
- 大气密度不确定性（200km高度可达±50%）
- 热效应与热控系统设计冲突
- 需要决策：是否考虑大气密度日变化和季节变化

## 3.2 大气密度模型选型

### 3.2.1 模型对比
- 指数模型：简单快速，适合算法原型
- NRLMSISE-00：高精度，包含太阳活动、地磁指数
- 需要决策：原型阶段精度与计算成本的权衡

### 3.2.2 推荐方案：分阶段采用
- 第一阶段：指数模型验证概念
- 第二阶段：NRLMSISE-00高保真仿真
- 需要决策：何时切换模型（基于轨道稳定性验证结果）

### 3.2.3 指数模型实现
- 参考高度密度ρ₀和标高H的取值
- 需要决策：使用标准大气值还是任务特定校准值

## 3.3 气动力计算模型

### 3.3.1 阻力公式
- 标准阻力公式F_drag = -½ρC_dAv²
- 需要决策：阻力系数C_d取值（2.2-2.5范围）

### 3.3.2 相对速度计算
- 考虑地球自转的大气共转效应
- 需要决策：是否忽略高层大气风场

### 3.3.3 升力效应（可选）
- 非球对称航天器可能产生升力
- 需要决策：是否在初步研究中忽略升力

## 3.4 热效应建模

### 3.4.1 气动加热率
- Sutton-Graves公式估算驻点热流
- 需要决策：热流计算精度要求（影响热防护设计）

### 3.4.2 表面温度估算
- 简化能量平衡方程
- 需要决策：是否进行详细热分析或简化处理

## 3.5 数值仿真实现方案

### 3.5.1 事件检测算法
- 检测大气穿越开始/结束高度阈值
- 需要决策：阈值高度设定（如200km±20km）

### 3.5.2 变步长积分策略
- 穿越期间减小步长保证精度
- 需要决策：最小步长限制（计算效率vs精度）

### 3.5.3 穿越过程高保真仿真
- 高时间分辨率记录穿越过程
- 需要决策：数据记录频率（每秒vs每0.1秒）

## 3.7 安全边界与风险评估

### 3.7.1 最小安全高度
- 防止轨道过快衰减导致再入
- 需要决策：安全高度余量（如150km vs 180km）

### 3.7.2 不确定性传播分析

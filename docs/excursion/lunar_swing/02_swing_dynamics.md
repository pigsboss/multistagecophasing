# 02. 摆动动力学理论：核心概念与算法选型

> **本阶段目标**：为共振摆动轨道提供足够的理论基础，明确关键动力学概念，并指导后续算法选型。本文档不涉及详细实现，而是作为后续开发的**理论指导手册**。

## 2.1 摆动轨道的物理图像

### 2.1.1 基本运动模式
地月共振摆动轨道是一种特殊的**三体周期轨道**，航天器在地球和月球之间形成类似"钟摆"的运动模式：

```
地球（近地点） ↔ 月球（远地点）
    ↓ 能量损失（大气阻力）    ↓ 能量增益（引力辅助）
    └─────────────────────────┘ 周期性平衡
```

### 2.1.2 关键轨道参数（设计变量）
在后续算法中，我们将搜索满足以下条件的轨道：
- **近地点高度**：h_p = 150-200 km（大气穿越高度）
- **远地点高度**：h_a ≈ 月球轨道距离（~380,000 km）
- **共振比**：n:m（如2:1, 3:2）
- **轨道倾角**：i = 5°-30°（形成三维摆动）

## 2.2 圆形限制性三体问题（CRTBP）框架

### 2.2.1 坐标系定义
我们的算法将在以下两个坐标系中工作：
1. **地月旋转坐标系** (`EARTH_MOON_ROTATING`)：用于CRTBP动力学计算
2. **地心惯性坐标系** (`J2000_ECI`)：用于高精度仿真和可视化

两个坐标系之间的转换通过 `math_tools.py` 中的 `rotating_to_inertial` 和 `inertial_to_rotating` 函数实现。

### 2.2.2 运动方程（无量纲形式）
在地月旋转坐标系中，无量纲化的运动方程为：
```
ẍ - 2ẏ = ∂Ω/∂x
ÿ + 2ẋ = ∂Ω/∂y
z̈ = ∂Ω/∂z
```
其中 Ω 为有效势能。

### 2.2.3 雅可比常数（能量不变量）
雅可比常数 C 是CRTBP中的运动不变量，定义为：
```
C = 2Ω(x,y,z) - (ẋ² + ẏ² + ż²)
```
**算法验证关键**：在无摄动的CRTBP积分中，C 值应保持不变（验证数值积分精度）。

## 2.3 共振条件与周期轨道

### 2.3.1 共振条件
n:m 共振轨道满足：
```
n·T_orbit = m·T_moon
```
其中 T_moon ≈ 27.3217 天（月球轨道周期）。

**常见共振比的选择依据**：
- **2:1共振**：轨道周期约13.66天，稳定性较高，作为首个验证目标
- **3:2共振**：轨道周期约18.21天，稳定性中等，作为备选目标

### 2.3.2 周期轨道条件
周期轨道满足边界条件：
```
x(T) = x(0), v(T) = v(0)
```
其中 T 为轨道周期（对n:m共振，T = m/n · T_moon）。

## 2.4 稳定性分析基础概念

### 2.4.1 状态转移矩阵（STM）
状态转移矩阵 Φ(t) 描述小扰动的传播：
```
δx(t) = Φ(t) · δx(0)
```
其中 Φ(t) 通过积分变分方程获得。

### 2.4.2 单值矩阵与稳定性
单值矩阵 M = Φ(T) 的特征值 λ_i 决定稳定性：
- 若所有 |λ_i| = 1，轨道线性稳定
- 若存在 |λ_i| > 1，轨道不稳定

**算法需求**：需要实现STM计算器 (`STMCalculator`)，用于稳定性分析和打靶法。

## 2.5 打靶法算法选型

### 2.5.1 单参数打靶法（首选）
针对单个周期轨道搜索：
1. **输入**：初始状态猜测 x₀，目标周期 T
2. **积分**：从 x₀ 积分时间 T 得到 x_T
3. **残差**：Ψ = x_T - x₀
4. **修正**：使用牛顿迭代法修正 x₀，直到 ||Ψ|| < ε

**优点**：实现简单，收敛快（对良好初始猜测）
**缺点**：对初始猜测敏感

### 2.5.2 多参数打靶法（备选）
将轨道分为多段，在各段连接处施加连续性条件，增加设计自由度以改善收敛性。

**适用场景**：当单参数打靶法难以收敛时使用。

## 2.6 初始猜测生成策略

### 2.6.1 基于二体近似的解析解（首选）
在二体近似下，共振轨道的轨道根数可解析计算：
1. 由共振比 n:m 计算轨道周期 T
2. 由近地点高度 h_p 计算半长轴 a 和偏心率 e
3. 假设小倾角，生成经典轨道根数
4. 转换到地月旋转坐标系

### 2.6.2 连续法（轨道族追踪）
从已知解（如平面轨道）出发，通过参数连续变化追踪轨道族。

## 2.7 摄动源优先级排序

在后续高保真仿真中，需按重要性顺序引入摄动：

| 优先级 | 摄动源 | 影响阶段 | 预期影响量级 |
|--------|--------|----------|--------------|
| 1 | 地球J2项 | 近地点 | 显著（~1e-3） |
| 2 | 地球高阶重力场（J2-J10） | 近地点 | 中等（~1e-5） |
| 3 | 大气阻力 | 近地点 | 显著（能量损失） |
| 4 | 月球非球形引力 | 远地点 | 小（~1e-6） |
| 5 | 太阳引力（第三体） | 全轨道 | 小（~1e-7） |

## 2.8 数值算法选型指南

### 2.8.1 积分器选择
1. **轨道搜索阶段**：固定步长Runge-Kutta 4/5（兼顾速度与精度）
2. **高保真仿真阶段**：变步长RKF78或DOP853（保证长期积分精度）

### 2.8.2 线性代数求解
1. **牛顿迭代**：使用 `numpy.linalg.solve` 或 `scipy.linalg.solve`
2. **病态情况**：使用SVD分解或正则化方法

### 2.8.3 事件检测
需要实现以下事件检测：
- **近地点穿越**（用于计算大气阻力）
- **远地点穿越**（用于分析月球引力效应）
- **庞加莱截面穿越**（用于稳定性分析）
